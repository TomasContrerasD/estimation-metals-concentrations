# estimation-metals-concentrations
Modelo de estimación de metales pesados en depósitos de relaves usando imágenes satelitales multiespectrales.

El modelo consiste en utilizar imágenes del satélite Landsat 8 para estimar las concentraciones de Cu, Ni, Zn, Co y As, en 228 depósitos de relaves en la región de Coquimbo, Chile. Para ello, se utilizan muestras tomadas por Sernageomin entre 2015 y 2017...
